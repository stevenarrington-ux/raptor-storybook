# Design Tokens

Design tokens are the atomic values that define Raptor's visual language. They live in two places:

- **`globals.css`** — CSS custom properties for direct use in components
- **`tokens.js`** — JavaScript exports for programmatic access

---

## Colors

### Primary — Raptor Orange

The primary palette drives all action states (buttons, links, focus rings, active navigation).

| Token | Hex | Usage |
| --- | --- | --- |
| `--color-primary-50` | `#FFF7ED` | Hover backgrounds, subtle fills |
| `--color-primary-100` | `#FFEDD5` | Selected row backgrounds |
| `--color-primary-200` | `#FED7AA` | Text selection highlight |
| `--color-primary-300` | `#FDBA74` | — |
| `--color-primary-400` | `#FB923C` | Active nav text, outline button border |
| `--color-primary-500` | `#F97316` | **Primary buttons, badge fills, focus rings** |
| `--color-primary-600` | `#EA580C` | Button hover state |
| `--color-primary-700` | `#C2410C` | Button active/pressed state |
| `--color-primary-800` | `#9A3412` | High-contrast text on primary-50 |
